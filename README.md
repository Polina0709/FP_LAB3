# Text Sentence Word Swapper (Haskell)

Цей проєкт призначений для обробки тексту з файлу:  
у **кожному реченні** перше слово міняється місцями з останнім, при цьому  
**зберігається пунктуація, пробіли, лапки, коми та структура речення**.

Проєкт виконаний на мові **Haskell**, з використанням інструменту **Stack** та розширеного парсера речень.

---

## ✨ Можливості

- Заміна першого та останнього слова в кожному реченні
- Збереження усіх розділових знаків (`. , ! ? – : « » ' …`)
- Нормалізація пробілів (кілька пробілів → один)
- Підтримка українського тексту
- Обробка з файлу → запис результату в інший файл
